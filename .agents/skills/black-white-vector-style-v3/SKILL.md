# 黑白矢量风格 3.0

## 工作流

1. 先读取 [references/style-guide.md](references/style-guide.md)，以 2.0 的“粗、疏、收尖、断开、局部汇黑”为基线。
2. 保留主体身份、比例、姿态、视角和关键结构，只转换明暗与线条语言。
3. 先把画面概括成四级：纯白亮部、短而断开的过渡线、粗长收尖的暗部线、小范围不规则实黑的最深暗部；不要从细节纹理开始。
4. 保持 2.0 的粗线重量和疏朗节奏，只在暗部与中间调适量增加约 15%–25% 粗排线，同时把约 20%–35% 的非必要实黑面积拆回分离排线。不要把线条变细或变成密集微线。
5. 让阴影线沿体积弯曲；从暗到亮依次缩短、变疏、断开并消失。只在深暗区加入少量第二方向的交叉排线。
6. 使用支持参考图的图像生成或编辑能力时，从 `assets/` 选择 2–4 张与题材和密度最接近的 2.0 粗线参考。只迁移共同线条语言，不复制具体构图或对象。
7. 按风格指南的验收表检查。优先修正过大的实黑块，同时防止排线退化为发丝线、全域细密纹理、点描噪声或均匀网纹。

## 输出约束

- 默认使用纯黑和纯白、干净白底、清楚粗轮廓、大面积留白，不使用灰色填充或渐变。
- 将“矢量风格”理解为边缘利落、形状概括、缩放可读的视觉语言。若用户要求 SVG、EPS 或可编辑路径，另行矢量描摹或重建并验证路径。
- 最深暗部仍允许少量粗线局部粘连成不规则黑块，但块面应明显少于 2.0；大部分暗部继续保留可辨认的粗排线和白缝。
- 用户要求透明背景时，将白底转换为透明并清除灰白毛边。
