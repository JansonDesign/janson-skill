# 黑白矢量风格 3.0 规范

## 1. 核心定义

完整保留 2.0 的粗犷、干净、矢量化线条语言：使用粗重、两端收尖的黑色刻线塑造体积，暗部由长而粗的随形线组成，向亮部过渡时线条逐渐缩短、变疏、断开，最后消失为纯白。深暗区可加入有限交叉排线。

3.0 只做两个幅度克制的调整：把 2.0 中一部分非必要实黑块拆回粗排线，并在暗部和中间调略微增加排线数量。不得改变 2.0 的线条粗度、疏朗节奏、构图方式、留白比例和整体矢量感。

这不是细密铜版蚀刻、发丝钢笔画或全域纹理化。排线增加后仍须粗犷、干净整洁、边缘利落。

## 2. 线条语法

### 主线形态

- 使用中段饱满、两端自然收尖的梭形、刀刻形或泪滴形线条。
- 线条应有可见重量，避免等宽发丝线、针尖线和密集微线。
- 外轮廓清楚而稳定；内部暗部线可接近外轮廓的粗度，次要结构线明显更少、更轻。
- 保留轻微手刻不规则：粗细、长度、弧度和间距可自然变化，但边缘必须利落，无毛糙灰边。

### 密度与间距

- 保持 2.0 的疏朗粗线节奏，只在暗部和中间调增加约 15%–25% 排线，不做全面增密。
- 增加数量时保持原有线条粗度，并同步保留清楚白缝；禁止为了容纳更多线而把线条变细。
- 中间调的白色间隙通常保持约为线条宽度的 1–2 倍；靠近亮部扩大到 3 倍以上或直接停止排线。
- 同一区域的线条不应等长、等距、机械平行；用长短错落和断续形成节奏。
- 第一印象应是粗犷黑白层次与有力排线，而不是大片黑面，也不是很多细线。

### 暗到亮的过渡

按以下顺序组织，不跳级：

1. 最深暗：粗长收尖线加有限交叉排线；允许少量线条局部粘连成小型不规则实黑，但大部分区域仍可辨认排线。
2. 深暗：粗长收尖线为主，比 2.0 略多，允许少量第二方向交叉排线加强体积。
3. 中间调：减少交叉线，保留单方向随形粗线；线条开始缩短、拉开。
4. 过渡亮部：使用更短、更疏、断开的收尖线或线段簇。
5. 高光：保持纯白，不填纹理，不以点描补满。

### 交叉排线与实黑块

- 交叉排线只放在最暗约 10%–20% 的区域，用一个辅助方向即可；不要编织成均匀网格。
- 交叉线也必须粗、收尖并随结构变化，不能退化为细密网纹。
- 实黑块仍应像多根粗刻线在最深暗处自然汇聚，边缘有凹凸、缺口和尖端，不做几何规则色块。
- 相比 2.0，把约 20%–35% 的非必要实黑面积拆回分离的粗排线或粗交叉排线；瞳孔、缝隙、深孔和接触遮挡点等必要实黑可保留。
- 不追求“完全没有黑块”。只要求黑块更少、更小，排线稍多，同时保持 2.0 的强对比和粗犷感。

## 3. 明暗与造型

- 先确定单一、清楚的主光方向，再分配白、线、黑三个大关系。
- 排线沿体积走向：球体用弧形线，圆柱顺轴或绕体，叶片沿叶脉，毛发随生长方向，布料随褶皱。
- 把照片式连续灰阶压缩成清楚层级；只比 2.0 略微增加排线层次，不追求细腻灰阶。
- 亮部至少保留一个完整、连贯的白色形状；不能被新增排线切散。
- 结构和识别优先于装饰。五官、关节、叶脉、接缝等关键线只保留必要部分。

## 4. 构图与题材适配

- 单体：完整、居中或轻微偏中，四周留约 15%–25% 白色空间；落地阴影主要用断开的粗排线，小黑块只作强调。
- 人像：把粗线集中在发际、眼窝、鼻底、下颌和衣褶暗部；脸部亮面保持大白区，用略多排线替换非必要黑面。
- 动物：先读剪影和物种特征；毛发用成组粗短线，背光面以稍多排线为主，仅局部汇成小黑块。
- 植物与食物：用叶脉、沟槽和生长方向组织粗弧线；高光区直接留白，暗侧线条逐步缩短断开。
- 建筑与器物：保留必要结构轮廓，省略微小砖纹和装饰；凹槽、窗洞和背光面可用略多粗排线，深缝保留小黑块。
- 场景：减少背景纹理，避免全幅排线；用大白区隔开主体，以局部粗排线和少量小黑块建立前后层次。

## 7. 失败信号与修正

- 线条太细：将主要阴影线加粗约 1.5–2.5 倍，删除次要微线，恢复 2.0 的粗线重量。
- 排线太密：只删除新增的多余排线并扩大白缝，不改变 2.0 的主要线条粗度。
- 排线仍太少：只在暗部和中间调补充约 15%–25% 同粗度随形线，不做全画面增密。
- 过渡生硬：按“长粗—短疏—断线—留白”重新组织，不用灰色渐变补偿。
- 黑块太多：把约 20%–35% 非必要黑面拆为分离粗排线或粗交叉排线，保留必要的小黑块。
- 黑块太少、对比不足：只在真正最深暗处恢复小范围不规则实黑，不扩大成大片黑面。
- 画面太脏：删除点描、纸纹、散乱碎线和背景噪声，保持白底纯净。

## 8. 验收表

- [ ] 整体第一眼仍与 2.0 相同：粗犷、干净整洁、矢量风格化。
- [ ] 主要阴影线明显粗重、中段饱满、两端收尖，没有变成细密微线。
- [ ] 排线比 2.0 略多，但只集中在暗部和中间调，白缝与大面积留白仍清楚。
- [ ] 实黑块比 2.0 略少、略小，但最深暗仍保留必要的小范围黑色强调。
- [ ] 暗到亮遵循长粗、缩短、变疏、断开、消失的顺序。
- [ ] 交叉排线只存在于深暗区，且没有形成均匀细密网格。
- [ ] 线条沿体积或材质方向变化，主体缩小后仍清楚可辨。
- [ ] 边缘干净、黑白纯粹，无灰雾、渐变、纸纹、毛边、文字或水印。
- [ ] 若为改绘，主体身份、比例、姿态与关键结构仍然准确。
- [ ] 若声称真矢量，交付物确为可编辑路径并通过小尺寸检查。
