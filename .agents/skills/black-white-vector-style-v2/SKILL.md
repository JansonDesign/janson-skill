# 黑白矢量风格 2.0

## 工作流

1. 先读取 [references/style-guide.md](references/style-guide.md)，把“粗、疏、收尖、断开、汇黑”作为硬约束。
2. 保留主体身份、比例、姿态、视角和关键结构，只转换明暗与线条语言。
3. 先把画面概括成四级：纯白亮部、短而断开的过渡线、粗长收尖的暗部线、不规则实黑的最深暗部；不要从细节纹理开始。
4. 主动减少线条数量并增加单线重量。若结果像细密铜版蚀刻，优先把线条数量减少约 40%–60%，把主要阴影线加粗约 1.5–2.5 倍，再重新生成。
5. 让阴影线沿体积弯曲；从暗到亮依次缩短、变疏、断开并消失。只在深暗区加入少量第二方向的交叉排线。
6. 使用支持参考图的图像生成或编辑能力时，从 `assets/` 选择 2–4 张最接近题材的粗线参考。只迁移共同线条语言，不复制具体构图或对象。
7. 按风格指南的验收表检查；发现发丝线、全域细密排线、点描噪声或均匀网纹时必须继续修正。

## 输出约束

- 默认使用纯黑和纯白、干净白底、清楚粗轮廓、大面积留白，不使用灰色填充或渐变。
- 将“矢量风格”理解为边缘利落、形状概括、缩放可读的视觉语言。若用户要求 SVG、EPS 或可编辑路径，另行矢量描摹或重建并验证路径。
- 最深暗部允许粗线粘连、汇聚成有机不规则实黑块；不要强行在所有黑块内部保留白缝。
- 用户要求透明背景时，将白底转换为透明并清除灰白毛边。
