# 黑白矢量风格 2.0 规范

## 1. 核心定义

使用少量、粗重、两端收尖的黑色刻线塑造体积。暗部由长而粗的随形线组成，向亮部过渡时线条逐渐缩短、变疏、断开，最后消失为纯白。深暗区可加入有限交叉排线；最深暗部让相邻粗线自然粘连、汇聚成不规则实黑块面。整体粗犷而不脏乱，边缘干净，具有明确的矢量图形感。

这不是细密铜版蚀刻、发丝钢笔画或全域纹理化。线条数量必须服从大明暗和缩略图识别度。

## 2. 线条语法

### 主线形态

- 使用中段饱满、两端自然收尖的梭形、刀刻形或泪滴形线条。
- 线条应有可见重量，避免等宽发丝线、针尖线和密集微线。
- 外轮廓清楚而稳定；内部暗部线可接近外轮廓的粗度，次要结构线明显更少、更轻。
- 保留轻微手刻不规则：粗细、长度、弧度和间距可自然变化，但边缘必须利落，无毛糙灰边。

### 密度与间距

- 默认“少线优先”：宁可扩大白区和黑块，也不要用大量细线模拟灰阶。
- 中间调的白色间隙至少约为线条宽度的 1–2 倍；靠近亮部时扩大到 3 倍以上或直接停止排线。
- 同一区域的线条不应等长、等距、机械平行；用长短错落和断续形成节奏。
- 若放大后第一印象是“很多细线”，判定失败；第一印象应是清楚的黑白块面与少量有力刻线。

### 暗到亮的过渡

按以下顺序组织，不跳级：

1. 最深暗：粗线彼此靠拢、粘连并形成边缘不规则的实黑块。
2. 深暗：粗长收尖线为主，允许少量第二方向交叉排线加强体积。
3. 中间调：减少交叉线，保留单方向随形粗线；线条开始缩短、拉开。
4. 过渡亮部：使用更短、更疏、断开的收尖线或线段簇。
5. 高光：保持纯白，不填纹理，不以点描补满。

### 交叉排线与实黑块

- 交叉排线只放在最暗约 10%–20% 的区域，用一个辅助方向即可；不要编织成均匀网格。
- 交叉线也必须粗、收尖并随结构变化，不能退化为细密网纹。
- 实黑块应像多根粗刻线在最深暗处自然汇聚，边缘有凹凸、缺口和尖端，不做几何规则色块。
- 允许实黑块承担瞳孔、缝隙、遮挡、物体背光面和结构深槽；避免无依据地把主体大面积涂黑。

## 3. 明暗与造型

- 先确定单一、清楚的主光方向，再分配白、线、黑三个大关系。
- 排线沿体积走向：球体用弧形线，圆柱顺轴或绕体，叶片沿叶脉，毛发随生长方向，布料随褶皱。
- 把照片式连续灰阶压缩成清楚层级；不要试图用更多线条还原每一级灰度。
- 亮部至少保留一个完整、连贯的白色形状；不能被零碎纹理切散。
- 结构和识别优先于装饰。五官、关节、叶脉、接缝等关键线只保留必要部分。

## 4. 构图与题材适配

- 单体：完整、居中或轻微偏中，四周留约 15%–25% 白色空间；落地阴影只用少量断线或小黑块。
- 人像：把粗线集中在发际、眼窝、鼻底、下颌和衣褶暗部；脸部亮面保持大白区，禁止用细线铺满皮肤。
- 动物：先读剪影和物种特征；毛发用成组粗短线，背光面可汇成不规则黑块。
- 植物与食物：用叶脉、沟槽和生长方向组织粗弧线；高光区直接留白，暗侧线条逐步缩短断开。
- 建筑与器物：保留必要结构轮廓，省略微小砖纹和装饰；用较大黑白面表现凹槽、窗洞和背光面。
- 场景：减少背景纹理，避免全幅排线；用大白区隔开主体，用局部黑块建立前后层次。

## 6. 可复用提示词

### 中文

## 7. 失败信号与修正

- 线条太细：将主要阴影线加粗约 1.5–2.5 倍，删除次要纹理线。
- 线条太密：删除约 40%–60% 排线，扩大白缝，让暗部直接合并为局部黑块。
- 过渡生硬：按“长粗—短疏—断线—留白”重新组织，不用灰色渐变补偿。
- 黑块太平：用收尖线端和不规则缺口打破边缘，使其像粗线自然汇聚。
- 黑块太多：恢复连续白色亮面，只在真正最深暗处保留实黑。
- 画面太脏：删除点描、纸纹、散乱碎线和背景噪声，保持白底纯净。

## 8. 验收表

- [ ] 第一眼看到的是清楚黑白块面，而不是密集细线。
- [ ] 主要阴影线明显粗重、中段饱满、两端收尖。
- [ ] 暗到亮遵循长粗、缩短、变疏、断开、消失的顺序。
- [ ] 亮部包含完整的大面积纯白形状，没有被微线或点描填满。
- [ ] 交叉排线只存在于深暗区，且没有形成均匀细密网格。
- [ ] 最深暗部可见粗线自然粘连成不规则实黑块，而非规则平涂色块。
- [ ] 线条沿体积或材质方向变化，主体缩小后仍清楚可辨。
- [ ] 边缘干净、黑白纯粹，无灰雾、渐变、纸纹、毛边、文字或水印。
- [ ] 若为改绘，主体身份、比例、姿态与关键结构仍然准确。
- [ ] 若声称真矢量，交付物确为可编辑路径并通过小尺寸检查。
